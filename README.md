Student Data Cleaning and Visualization Project

Project Overview

This project focuses on cleaning, preprocessing, analyzing, and visualizing a student dataset using Python. The goal is to transform raw data into meaningful insights through data cleaning and exploratory data analysis (EDA).

Dataset Description

The dataset contains information about 100 students with the following attributes:

- Student ID
- Name
- Gender
- Age
- Department
- Attendance (%)
- Marks
- City

Data Cleaning Steps

The following data cleaning operations were performed:

- Removed duplicate records
- Handled missing values
- Standardized categorical values
- Corrected inconsistent text formats
- Detected and treated outliers
- Converted data types where necessary

Feature Engineering

New features were created from existing data:

- Grade
- Result (Pass/Fail)
- Attendance Status
- Scholarship Eligibility
- Age Group

Exploratory Data Analysis

The following analyses were performed:

- Student distribution by department
- Gender distribution
- City-wise student distribution
- Marks distribution
- Attendance analysis
- Correlation analysis

Visualizations

The project includes:

- Bar Charts
- Pie Charts
- Histograms
- Scatter Plots
- Correlation Heatmaps

Key Insights

- Students with higher attendance generally achieved higher marks.
- CSE department had the highest student count.
- Most students belonged to the 20–21 age group.
- Attendance showed a positive relationship with academic performance.

Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
